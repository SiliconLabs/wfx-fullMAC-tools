# WF200 FMAC tools

This repository contains tools and examples to use the **WF200 FMAC driver**.

The WF200 is a WiFi transceiver. See [a link]https://www.silabs.com/products/wireless/wi-fi/wi-fi-transceivers/device.wf200 for details.

The WF200 FMAC driver documentation is located at: https://docs.silabs.com/wifi/wf200/rtos/0.1/index

The WF200 FMAC driver standalone repository is at [a link]https://github.com/SiliconLabs/wfx-fullMAC-driver
